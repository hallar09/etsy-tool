# My Etsy Shop Manager

A **private internal tool** for managing my personal Etsy shop(s).

## Purpose
This application helps me efficiently operate my Etsy business by:
- Automatically updating product listings (prices, inventory, descriptions)
- Retrieving and analyzing order/sales data
- Generating internal performance reports

## Features
- Read/write access to my own shop listings
- Order data synchronization
- Secure local/server-based operation
